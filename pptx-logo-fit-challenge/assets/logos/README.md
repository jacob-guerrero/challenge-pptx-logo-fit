# Reference logos

Place these three files here (generated by `npm run setup` or your own artwork):

| File | Role |
|------|------|
| `square.png` | Square logo used for `output/branded_square.pptx` |
| `h_rectangle.png` | Horizontal logo for `output/branded_h_rectangle.pptx` |
| `v_rectangle.png` | Vertical logo for `output/branded_v_rectangle.pptx` |

The baseline script does **not** fix aspect ratio — that is what candidates must solve.
